What does "building" mean?
----------------------------
Your C# code is just text; it needs to be converted into a format that the computer can understand.
The dotnet build command compiles (translates) your C# code into DLL (Dynamic Link Library) files that can be executed.

How to Build an ASP.NET Project?
----------------------------------
In your terminal or command prompt, navigate to your project folder and run:

	" dotnet build "
